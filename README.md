## 🏦 Loan Approval Prediction ML Pipeline

A machine learning project built using Python and Jupyter Notebook that predicts loan approval status based on applicant information such as income, credit history, education, employment status, and loan amount. The project demonstrates a complete ML pipeline including data preprocessing, feature engineering, model training, prediction, and evaluation using real-world financial datasets.

## 🚀 Features
* **Loan Approval Prediction System** – Predict whether a loan application will be approved or rejected.
* **Complete ML Pipeline** – Covers preprocessing, training, testing, and evaluation workflows.
* **Data Cleaning & Preprocessing** – Handle missing values and inconsistent records.
* **Feature Engineering** – Prepare financial and applicant features for modeling.
* **Exploratory Data Analysis (EDA)** – Analyze applicant and loan trends visually.
* **Categorical Data Encoding** – Transform non-numeric variables for ML models.
* **Model Training & Evaluation** – Train classification models and evaluate performance.
* **Prediction Analysis** – Generate loan approval predictions from applicant data.
* **Visualization & Insights** – Create charts and graphs for business understanding.
* **Real-World Financial Dataset** – Practice machine learning using banking-related data.

## 🛠️ How to Run
1. **Clone the repository**:
```
git clone https://github.com/Agent-A345/Loan-Approval-ML-Pipeline.git
```
2. **Navigate to the project folder**:
```
cd Loan-Approval-ML-Pipeline
```
3. **Install required libraries**
```
pip install pandas numpy matplotlib seaborn jupyter scikit-learn
```
4. **Run the notebook**
```
jupyter notebook
```
5. **Open**
```
loan_approval_ml_pipeline.ipynb
```
